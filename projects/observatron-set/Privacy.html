Observatron Privacy Policy

Shiresmith Publishing does not collect or store your data, period. The only data Observatron stores is high scores, and that is stored only on the device.

In the event of an error that crashes the app, we will collect data to help debug the problem. This data includes the operating system version and log data about the area of the code that was executing when the error occurred. There is no personal information included in any error report.
